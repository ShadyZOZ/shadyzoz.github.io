转接google二维码api

  • PHP
  • ToolBox

新版微信客户端支持了二维码解析，觉得是时候在微信平台里做一个生成二维码的api了（其实是放假无聊，自己脑洞大开=_=）

太懒，直接用google的api，放到日本的空间用的灰常开心，到阿里云就不开心了。好吧，哪就基于google的自己再写一个api放在DO上好了。。。。。。

正好打算自己写一个工具集。

google chart

google文档 需要科学上网才能看。。

其实很简单

https://chart.googleapis.com/chart?chs=150x150&cht=qr&chl=Hello%20world

摘自文档

  • https://chart.googleapis.com/chart? - All infographic URLs start with this root URL, followed by one or more parameter/value pairs. The required and optional parameters are specific to each image; read your image documentation.

  • chs - Size of the image in pixels, in the format widthxheight

  • cht - Type of image: ‘qr’ means QR code.
  • chl - The data to encode. Must be URL-encoded.

简单翻译。。

  • https://chart.googleapis.com/chart? - 请求的根地址，跟随不同参数，有不同的效果哦～详情咨询阅读文档。。

  • chs - 控制图像大小，形式：widthxheight

  • cht - 图像类型，qr代表二维码
  • chl - 需要加密的内容

shady api

很简单，用get传递参数，然后将生成内容和图像大小作为一个字符串进行md5加密，命名png文件,存到本地。

api地址：http://tools.shadyzoz.com/qr/qr_google.php?

参数

  • widhtHeight - 图片长宽
  • chl - 内容

qr_google.php

示例

请求地址： http://tools.shadyzoz.com/qr/qr_google.php?widhtHeight=300&chl=hello

二维码库： http://tools.shadyzoz.com/qr/db/47d529b39d63df9f74aec6f33bab4fb4.png
